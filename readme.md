## TODO Before Run

1. Fetch dependencies

```sh
go mod download
```

2. Install ginkgo for run all test suites

```sh
go install github.com/onsi/ginkgo/v2/ginkgo
```

- Run the main function

```sh
go run codewars.go
```

- Run all tests

```sh
ginkgo ./...
```
